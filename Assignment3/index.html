Requirement 3

Coming into this assignment, I already had an idea of how I would complete it. The only challenge for me was the javascript script, but I came to the conclusion it's not so bad.

To replicate my results, you may:
  • Access the javascript script from the navigation tab at the top left.
  • Use the script as a reference, with some help from the web, to build a web page.
  • Customize the layout and colors using CSS to your desired taste.
  • Integrate the script into the HTML to make the web page functional.
  • Enhance your web design skills by watching YouTube videos and following someone's code.
  • Use this exercise as an opportunity to pick up on any code or material that you may have forgotten or can't remember from the past.
  • Enjoy the result of your hard work and be pleased with the outcome.

Requirement 4

HTML played a crucial role in the implementation of my temperature converter. It provided the user input field through the use of the (div) HTML code, without which the JavaScript script would have been insufficient. The JavaScript script in turn generated the output on the screen after the user input was processed. I initially believed that a server-side technology was necessary for any implementation with input and output, but this was purely client-side. CSS added an aesthetic touch, bringing colors and life to the pages. I especially loved my dark theme version of the Temperature Converter.
